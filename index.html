Sitio Web HelpDesk

He desarrollado un sitio web estatico que combina diversas tecnologías web, como HTML, CSS, Bootstrap y JavaScript para crear una experiencia atractiva y funcional. El sitio ofrece valiosos recursos en formato PDF con instructivos de autogestion.

Además, he incorporado un formulario de asistencia técnica que se envía automáticamente a un archivo de Google Sheet mediante JavaScript y Google Apps Script. Esto permite una gestión eficiente de los problemas técnicos y una respuesta rápida a las solicitudes de soporte.

El objetivo principal del sitio web es brindar apoyo y asistencia técnica a los miembros del personal de la empresa, mejorando así su experiencia y productividad en el trabajo. El sitio cuenta con un sistema básico de protección que requiere una única contraseña para acceder.

Es importante destacar que este repositorio de ejemplo no contiene todos los recursos e instructivos completos, ya que algunos elementos son exclusivos de la empresa y no están disponibles públicamente. Sin embargo, el repositorio proporciona una visión general de la estructura y las tecnologías utilizadas en el desarrollo del sitio web.
